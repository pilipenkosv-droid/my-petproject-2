# Система «Антиплагиат» в РФ: как работает, откуда берёт данные и что безопасно для студентов

## 1. Что такое система «Антиплагиат»

Система «Антиплагиат» — это российский программно-аппаратный комплекс, разработанный компанией Forecsys и рекомендованный Рособрнадзором с 2007 года. Существует в нескольких версиях:

- **Антиплагиат.ру** (antiplagiat.ru) — открытый бесплатный сервис для всех пользователей. Проверяет только по одному модулю — открытые источники интернета.
- **Антиплагиат.ВУЗ** — закрытая платная версия для учебных заведений. Содержит 30–37+ модулей поиска, включая закрытые базы. Именно эта версия используется преподавателями при официальной проверке.
- **Антиплагиат (платный тариф для частных лиц)** — промежуточный вариант с расширенным набором модулей, но не таким полным, как у ВУЗа.

## 2. Модули поиска: откуда берутся данные

Ключевое отличие версии «Антиплагиат.ВУЗ» от бесплатной — доступ к закрытым коллекциям (модулям). Каждый ВУЗ при покупке лицензии выбирает набор модулей. Основные из них:

### Базовые модули (есть практически у всех ВУЗов)

| Модуль | Что содержит |
|--------|-------------|
| **Модуль поиска Интернет** | Открытые сайты, проиндексированные системой |
| **Кольцо вузов** | Все студенческие работы (ВКР, курсовые, дипломы), загруженные преподавателями ВУЗов-участников проекта после защиты |
| **Коллекция РГБ** | Авторефераты, кандидатские и докторские диссертации из Российской государственной библиотеки |
| **Модуль «Цитирование»** | Позволяет отличить правильно оформленные цитаты от плагиата |
| **Модуль поиска перефразирований Интернет** | Обнаруживает рерайт текстов из интернет-источников (с 2018 г.) |
| **Модуль поиска вуза** (внутренний) | Собственная база конкретного университета — все работы студентов данного ВУЗа |

## 3. Как работает алгоритм

Система использует несколько технологий:

**Алгоритм шинглов** — текст разбивается на перекрывающиеся последовательности слов (шинглы). Стандартный шингл — это цепочка из нескольких слов. Система создаёт хеш каждого шингла и сравнивает с базой. Чем короче шингл, тем строже проверка — модуль перефразирований использует более короткий шаг.

**NLP и нейросети** — современные модули используют технологии обработки естественного языка для выявления перефразирований, переводов и замен синонимами.

**Детектор ИИ-текста** — с недавних пор система определяет текст, сгенерированный нейросетями (ChatGPT и др.) и помечает работу знаком «Подозрительный документ: сгенерированный текст».

**Детектор технических манипуляций** — выявляет замену символов (кириллица ↔ латиница), вставку невидимого текста, скрытые символы, макросы.

## 4. Главный вопрос: попадает ли работа в базу при проверке?

Это центральная проблема, описанная в вопросе. Ответ зависит от того, ГДЕ именно происходит проверка.

### 4.1. Антиплагиат.ру (бесплатный / платный для частных лиц)

Согласно официальной позиции системы: **«При проверке документов в системе "Антиплагиат" для частных клиентов тексты работ системой не индексируются, в дальнейшем поиске на наличие заимствований участия не принимают и доступны только лицу, загрузившему документ»**.

Это значит, что при проверке через antiplagiat.ru (личный кабинет обычного пользователя) работа формально **не должна** попасть в базу.

### 4.2. Антиплагиат.ВУЗ (корпоративный аккаунт учебного заведения)

Здесь ситуация принципиально другая:

- **Простая проверка** преподавателем — работа проверяется, но **не индексируется автоматически** в «Кольцо вузов».
- **Добавление в индекс** — это **отдельное действие**, которое преподаватель совершает сознательно (кнопка «Добавить в индекс» / «Добавить в Кольцо вузов»). Обычно это делается в конце учебного года после защиты.
- **Внутренний модуль вуза** — работа, загруженная в систему конкретного ВУЗа, видна при проверках в рамках этого ВУЗа.

### 4.3. Почему тогда появляется проблема «100% заимствований»?

Есть несколько сценариев, приводящих к этому:

**Сценарий 1: Знак «Проверен в другой организации» (дубликат)**

Это относительно новая функция. Система Антиплагиат.ВУЗ теперь **отслеживает факт проверки документа** через корпоративные аккаунты. Если вы попросили знакомого сотрудника другого ВУЗа проверить работу через их корпоративный аккаунт Антиплагиат.ВУЗ — при повторной проверке в вашем ВУЗе появится предупреждение: «Есть подозрения на следующие группы маскировки заимствований: проверка в другой организации».

Это **не 100% плагиата**, а именно предупреждающий знак. Но он привлекает внимание преподавателя и может создать проблемы.

**Сценарий 2: Сторонний сервис с доступом к Антиплагиат.ВУЗ**

Многочисленные коммерческие сервисы (сайты-посредники) предлагают проверку «как в ВУЗе» за деньги. Они используют корпоративные аккаунты реальных учебных заведений. Проверка через такой сервис эквивалентна проверке через чужой ВУЗ — и система может пометить работу.

Более того, по мнению опытных пользователей: **некоторые платные сервисы АПВУЗ могут индексировать работу** или каким-то образом снижать оригинальность при повторной проверке. Это не подтверждено официально, но является устойчивым наблюдением сообщества.

**Сценарий 3: Работа уже есть в базе**

Если работа (или значительная её часть) ранее была загружена кем-то — другим студентом, куплена с сайта готовых работ, или текст был опубликован в интернете — она уже может быть в базе.

**Сценарий 4: Накопительный эффект**

Даже если работу не индексировали целиком, отдельные фрагменты могут совпадать с множеством источников: нормативные акты, устоявшиеся научные формулировки, цитаты, общеупотребительные фразы — всё это в сумме может дать высокий процент совпадений.

## 5. «Безопасный контур» для студентов

### ✅ БЕЗОПАСНО — не повлияет на оригинальность

| Сервис / Метод | Почему безопасно |
|----------------|-----------------|
| **Text.ru** | Полностью сторонняя система, проверяет только по интернету. Не связана с Антиплагиат. Работы не передаются в базы ВУЗов |
| **eTXT Антиплагиат** | Копирайтерский сервис, проверяет только открытые веб-страницы. Никак не связан с Антиплагиат.ВУЗ |
| **Advego Plagiatus** | Аналогично — сторонний копирайтерский сервис |
| **РУКОНТЕКСТ (text.rucont.ru)** | **Прямо заявляет**: «Проверяемые через систему документы в базу индексов не добавляются. Факт проверки документа не влияет на показатель его оригинальности в будущем» |
| **Plagiat.AI** | Бесплатный сервис на нейросетях. Заявляет: «Проверенные файлы не попадают в общую базу Антиплагиат ВУЗ, после проверки не хранятся на серверах» |
| **Антиплагиат.ру (бесплатная версия, личный кабинет)** | По официальной позиции — работы частных пользователей не индексируются. Но: проверка ведётся только по 1 модулю (интернет), результат будет кардинально отличаться от ВУЗовского |
| **Проверка средствами своего ВУЗа** (если ВУЗ даёт студентам попытки) | Некоторые ВУЗы (например, НИУ ВШЭ) дают студентам ограниченное число проверок (15 в год). Это безопасно, т.к. ВУЗ контролирует процесс |

### ⚠️ РИСК — может повлиять на проверку

| Сервис / Метод | В чём риск |
|----------------|-----------|
| **Проверка через знакомого сотрудника ДРУГОГО ВУЗа** | Работа проверяется через корпоративный аккаунт другого учебного заведения → появится знак «Проверен в другой организации». Это не снижает оригинальность, но привлекает внимание |
| **Платные сервисы-посредники с доступом к Антиплагиат.ВУЗ** | Используют корпоративные аккаунты реальных ВУЗов. Аналогичный риск знака «дубликата». Некоторые сервисы обещают «без следов», но гарантий нет |
| **Повторная проверка в своём ВУЗе** (если преподаватель проверяет несколько раз) | Если преподаватель не удалил первую проверку и работа попала во внутренний модуль ВУЗа — повторная проверка может показать совпадение с самой собой |

### 🚫 ОПАСНО — серьёзно повлияет

| Действие | Последствие |
|----------|-------------|
| **Публикация работы в интернете** (загрузка на сайты рефератов, форумы, облачные хранилища с публичным доступом) | Система проиндексирует текст из интернета и найдёт 100% совпадение |
| **Продажа / передача работы другому студенту** (если тот сдаст раньше) | Работа будет проиндексирована при защите первого студента, второй получит 100% плагиата |
| **Использование готовых купленных работ** | Скорее всего уже есть в «Кольце вузов» или открытых источниках |
| **Кодирование / техническое повышение оригинальности** | Система определяет такие манипуляции и ставит знак «Подозрительный документ» |
| **Генерация текста нейросетями без переработки** | Модуль определения ИИ-текста пометит работу |

## 6. Практические рекомендации

**Для предварительной проверки используйте:**
1. Бесплатные сторонние сервисы (Text.ru, eTXT) — для общей оценки уникальности по интернету
2. РУКОНТЕКСТ — если нужна проверка ближе к вузовской, с гарантией, что работа не попадёт в базу
3. Средства своего ВУЗа — если есть такая возможность (как в НИУ ВШЭ)

**Избегайте:**
1. Проверок через корпоративные аккаунты других ВУЗов (даже через знакомых)
2. Сомнительных платных сервисов-посредников — если не уверены, что они не оставляют следов
3. Загрузки черновиков в любые публичные места

**Помните:** бесплатные сервисы покажут результат, сильно отличающийся от вузовского (обычно в лучшую сторону), потому что не имеют доступа к закрытым базам. Разница может составлять 30–50%. Если eTXT показывает 80% оригинальности — Антиплагиат.ВУЗ может показать 40–50%.

## 7. Итоговая схема

```
Работа студента
       │
       ├── Проверка через Text.ru / eTXT / Advego / РУКОНТЕКСТ
       │   → Безопасно. В базу Антиплагиата НЕ попадает.
       │   → Но результат не совпадёт с вузовским.
       │
       ├── Проверка через Антиплагиат.ру (свой аккаунт)
       │   → Условно безопасно. Формально не индексируется.
       │   → Показывает только по 1 модулю — ненадёжный ориентир.
       │
       ├── Проверка через средства СВОЕГО ВУЗа (если дают попытки)
       │   → Безопасно. Максимально близко к финальному результату.
       │
       ├── Проверка через знакомого из ДРУГОГО ВУЗа
       │   → ⚠ Появится знак «проверен в другой организации».
       │   → Не снижает оригинальность, но создаёт вопросы.
       │
       ├── Проверка через платный сервис-посредник АПВУЗ
       │   → ⚠ Может появиться знак дубликата.
       │   → Некоторые сервисы обещают «без следов» — верифицировать сложно.
       │
       └── Официальная проверка при сдаче
           → Проверяется по всем модулям ВУЗа.
           → После защиты преподаватель может добавить в «Кольцо вузов».
```
