# Журнал изменений

## Февраль 2026

### 5 новых инструментов (2026-02-08)

Реализована полная экосистема из 6 AI-инструментов для подготовки научных работ:

**Новые инструменты:**
- **Генератор плана** (`/outline`) — AI создаёт структуру работы с разделами, подразделами и рекомендуемым объёмом
- **Краткое содержание** (`/summarize`) — генерация аннотации с выбором длины (короткое/среднее/подробное), поддержка ввода текста и загрузки файла
- **Повышение уникальности** (`/rewrite`) — рерайт текста в 3 режимах глубины (лёгкий/средний/глубокий) с возможностью сохранения терминов
- **Проверка грамматики** (`/grammar`) — интеграция с LanguageTool API, подсветка ошибок по типам, поддержка загрузки файла
- **Подбор литературы** (`/sources`) — поиск через OpenAlex + CrossRef с AI-валидацией релевантности, форматирование по ГОСТ

**Новые API endpoints:**
- `POST /api/generate-outline` — генерация плана
- `POST /api/summarize` — краткое содержание
- `POST /api/rewrite` — рерайт текста
- `POST /api/check-grammar` — проверка грамматики
- `POST /api/find-sources` — подбор литературы
- `POST /api/extract-text` — извлечение текста из документа
