# Примеры форматирования сообщений об ошибках

Этот документ демонстрирует, как изменились сообщения об ошибках форматирования, чтобы они стали более понятными для пользователей.

## Выравнивание текста

### Было (технический формат):
```
Неверное выравнивание текста: ожидается "justify", найдено "center"
```

### Стало (человекочитаемый формат):
```
Неверное выравнивание текста: ожидается "по ширине", найдено "по центру"
```

---

## Размер шрифта

### Было:
```
Неверный размер шрифта: ожидается "14 pt", найдено "12 pt"
```

### Стало:
```
Неверный размер шрифта: ожидается "14 пт", найдено "12 пт"
```

---

## Межстрочный интервал

### Было:
```
Неверный межстрочный интервал: ожидается "1.5", найдено "1.0"
```

### Стало:
```
Неверный межстрочный интервал: ожидается "полуторный", найдено "одинарный"
```

---

## Типы кавычек

### Было:
```
Использованы прямые кавычки вместо угловых: ожидается "angular", найдено "straight"
```

### Стало:
```
Использованы прямые кавычки вместо угловых: ожидается "угловые кавычки «»", найдено "прямые кавычки """
```

---

## Типы тире

### Было:
```
Использовано запрещенное длинное тире: ожидается "en-dash", найдено "em-dash"
```

### Стало:
```
Использовано запрещенное длинное тире: ожидается "среднее тире (–) или дефис (-)", найдено "длинное тире (—)"
```

---

## Неразрывные пробелы

### Было:
```
Требуется неразрывный пробел перед единицей измерения: ожидается "nbsp", найдено "space"
```

### Стало:
```
Требуется неразрывный пробел перед единицей измерения: ожидается "неразрывный пробел", найдено "обычный пробел"
```

## Реализация

Все преобразования реализованы в файле `src/lib/utils/formatting-messages.ts` с помощью функции `formatViolationMessage()`, которая автоматически определяет тип правила по `ruleId` и применяет соответствующее форматирование.

### Поддерживаемые типы правил:

- `*-align-*` - выравнивание текста
- `*-font-size-*` - размер шрифта
- `*-font-*` - семейство шрифта
- `*-spacing-*` - межстрочный интервал
- `*-indent-*` - отступы
- `*-margin-*` - поля страницы
- `*-quotes-*` - кавычки
- `*-dash-*` - тире
- `*-nbsp-*` или `*-space-*` - пробелы
