## TL;DR (главное за 30 секунд)

1. **Google за 28 дней: 13 кликов и 1491 показ** по всему домену. Катастрофически мало.
2. **Гипотеза пользователя подтверждена, но сильнее ожидаемого**: 0 (ноль) пересечений топ-Yandex-семантики с тем, что сейчас ранжируется в Google. Мы продвигаем в Яндексе "проверка пунктуации" (211K), "перефразировать текст" (109K) — в Google НИ ОДИН из топ-Wordstat-запросов не генерит нам показов.
3. **69% сайта вне индекса Google**: из 91 URL в sitemap проиндексировано только 28. Пять критичных страниц (все tool-страницы + главные лендинги) — **Google не знает о них**.
4. **Контент-ловушка**: 74% всех показов в Google (1105/1491) приходятся на ОДИН пост — `/blog/gost-r-7-0-11-2011-trebovaniya-k-dissertaciyam`, при этом CTR = **0.09%** (1 клик на 1105 показов). Google показывает нас, но никто не кликает. Плюс это не наша ЦА — это запросы про диссертации (PhD), не про студентов.
5. **Google-SERP по нашим коммерческим запросам занят контент-фермами и устоявшимися продуктами** (zaochnik.ru, gostdiploma.ru, text.ru, LanguageTool, retext.ai, studgen.ru, VC.ru/DTF/Habr рейтинги). Diplox отсутствует в топ-10 по всем проверенным целевым запросам.

**Вывод:** В Google мы сейчас — "мёртвый" сайт для коммерческих запросов. Это не вопрос позиций 11-20, это вопрос отсутствия в индексе + отсутствия доверия у Google (нет ссылок, свежий домен, тонкие сигналы E-E-A-T).

## 3. CTR-аномалия: gost-r-7-0-11-2011

Один блог-пост делает 74% показов всего сайта, но даёт 1 клик.

**Причины низкого CTR (гипотезы):**
1. **Запрос не матчится с заголовком.** Люди ищут "ГОСТ Р 7.0.11-2011 оформление курсовой работы", но наш пост, судя по slug, про "требования к диссертациям". Google показывает как релевантное, пользователь в сниппете видит "диссертация" → пролистывает дальше.
2. **Конкуренция в топ-10 по этому запросу.** Там — сайты с прямым матчем "оформление курсовой по ГОСТ".
3. **Snippet некачественный** (надо глянуть meta description).
4. **Средняя позиция 6.8** — сниппет виден, но не на первых 3-х строках.

**Action:** либо переписать title/description под более широкий запрос, либо сделать отдельный пост "ГОСТ Р 7.0.11-2011 для курсовой работы".

## 6. Root-cause: почему Google не даёт трафик

Синтез четырёх слоёв:

### 6.1 Индексация (технический слой)
- **69% URL вне индекса** — базовая проблема. Нет индекса → нет показов.
- Домен diplox.online новый (прогресс с 0/43 до 28/91 за неделю — Google индексирует, но медленно).
- Критичные конверсионные страницы (`/grammar`, `/summarize`, `/outline`, лендинги `/referat`, `/esse` etc.) — не индексированы.
- Есть остаточный noindex в памяти Google для 3 страниц (бывший баг?).
- 1 страница 403 для Googlebot (`kak-povysit-antiplagiat-kursovoj`).

### 6.2 Авторитет / ссылочный (trust-слой)
- Google для свежего домена требует **ссылок и упоминаний** для ранжирования по коммерческим запросам. Яндекс — снисходительнее.
- Domain age, backlinks — не проверялись напрямую (нужен Ahrefs/Moz), но по отсутствию в топ-10 по любому коммерческому запросу очевидно: ссылочный профиль слабый.
- Нет упоминаний в VC/DTF/Habr рейтингах.

### 6.3 Семантика (content targeting)
- Keyword metadata, заголовки постов и title-теги писались под Wordstat (Яндекс). Google-специфичные формулировки не учтены:
  - Google часто любит более "вопросительные" и длинные запросы.
  - "2026" в заголовках — почти все конкуренты это делают, у нас частично.
  - Формулировки "пример/образец" — сильнее ранжируются в Google, чем "требования/правила".
- Нет таргетинга под конкретные intent-кластеры Google: "пример оформления X", "шаблон X docx", "бесплатно X онлайн".

### 6.4 E-E-A-T (quality signals)
- Нет публичной информации об авторе (student-сервис = YMYL-адъяцент: "образование").
- Нет `datePublished/dateModified` в schema.org для части постов (проверить надо).
- Нет социальных/авторитетных сигналов (upvotes, shares).
- Главная не продаёт "кто мы и почему нам верить" — бандит даёт trust только по brand-запросам.
