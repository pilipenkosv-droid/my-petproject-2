# Введение

Актуальность темы обусловлена стремительным развитием технологий искусственного интеллекта и их внедрением во все сферы деятельности человека. Обработка естественного языка является одной из ключевых задач современной информатики.

Цель работы — провести сравнительный анализ методов машинного обучения, применяемых для решения задач обработки текста на русском языке. Для достижения цели были поставлены задачи по изучению теоретической базы и проведению экспериментов.

## Объект и предмет исследования

Объектом исследования выступают алгоритмы машинного обучения. Предметом — их применимость к задачам классификации русскоязычных текстов различных жанров.

## Методы исследования

В работе применялись общенаучные методы анализа и синтеза, а также специализированные методы статистического анализа и машинного обучения.

# Глава 1. Анализ

В данной главе проводится обзор существующих подходов к решению задач обработки естественного языка. Рассматриваются как классические статистические методы, так и современные нейросетевые архитектуры.

Сумма весов признаков вычисляется по формуле $\sum_{i=1}^{n} x_i$, где $n$ — общее количество признаков в модели. Интегральная оценка качества модели может быть записана как:

$$\int_0^1 f(x)dx$$

## Сравнение алгоритмов

Ниже представлены результаты экспериментального сравнения трёх алгоритмов на контрольной выборке.

| Алгоритм         | Точность | F1-score |
|------------------|----------|----------|
| Naive Bayes      | 0.82     | 0.79     |
| SVM              | 0.88     | 0.86     |
| BERT (fine-tune) | 0.94     | 0.93     |

Как видно из таблицы, наилучшие результаты демонстрирует модель BERT.

## Основные этапы обработки

Процесс обработки текста включает следующие этапы:

- Токенизация исходного текста
- Удаление стоп-слов и нормализация
- Векторизация признаков
- Обучение модели классификатора

Порядок выполнения экспериментов:

1. Подготовка обучающей выборки
2. Настройка гиперпараметров моделей
3. Кросс-валидация и оценка качества
4. Анализ ошибок классификации
